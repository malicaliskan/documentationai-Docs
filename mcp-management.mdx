## MCP Documentation

Mindbricks provides MCP integration at two levels, each documented on its own page:

### [Application MCP](./application-mcp)

How generated Mindbricks projects expose their Business APIs as MCP tools through the MCP-BFF aggregation layer. Covers tool discovery, authentication, transport options, and frontend prompts for AI-driven frontends.

### [Genesis MCP](./genesis-mcp)

The Mindbricks Genesis MCP server — the platform-level MCP endpoint for AI agents to design, configure, and manage projects. Covers schema tools, project data tools, frontend prompt tools, and connecting external tools like Cursor and Claude Desktop.
